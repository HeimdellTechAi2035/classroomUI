# Week 1: Onboarding & Assessment

## Overview
Welcome to the RemoteAbility training programme! This first week is all about getting to know each other, understanding the programme structure, and completing initial assessments.

## Learning Outcomes
By the end of this week, trainees will:
- Understand the 6-week programme structure and expectations
- Have completed initial skills assessments
- Be set up with all required tools and access
- Have established personal learning goals
- Know the key contacts and support channels

## Session 1: Welcome & Induction (2 hours)

### Agenda
1. **Welcome & Introductions** (15 mins)
   - Trainer introduction
   - Programme overview
   - Icebreaker activity

2. **Programme Overview** (30 mins)
   - 6-week structure explained
   - What to expect each week
   - Assessment and progression

3. **Policies & Procedures** (20 mins)
   - Safeguarding awareness
   - Code of conduct
   - Attendance expectations

4. **Break** (15 mins)

5. **Initial Assessment** (30 mins)
   - Skills baseline quiz
   - Individual support needs discussion

6. **Wrap-up & Next Steps** (10 mins)
   - Questions
   - Homework assignment
   - Preview of Session 2

### Trainer Notes
- Create a welcoming, non-judgemental atmosphere
- Some trainees may be nervous - reassure them
- Have welcome packs ready before the session
- Note any accessibility needs mentioned
- If assessments reveal support needs, document discreetly

### Resources Needed
- Welcome Pack (PDF)
- Assessment Form (online)
- Programme Overview Slides

## Session 2: Getting Set Up (2 hours)

### Agenda
1. **Review & Check-in** (10 mins)
   - How are people feeling?
   - Any questions from Session 1?

2. **Technology Setup** (40 mins)
   - Setting up required accounts
   - Navigating key tools
   - Troubleshooting common issues

3. **Communication Skills Intro** (30 mins)
   - Professional communication basics
   - Email etiquette overview

4. **Break** (15 mins)

5. **Goal Setting Workshop** (20 mins)
   - Setting SMART goals
   - Individual goal planning

6. **Week 1 Summary** (5 mins)
   - Review of key learnings
   - Preview Week 2

### Trainer Notes
- Be patient with technology setup - go at the slowest person's pace
- Have backup plans if technology fails
- Celebrate small wins
- Make sure everyone has working access before ending

## Common Questions

**Q: What if I miss a session?**
A: Contact your trainer as soon as possible. We can arrange catch-up support.

**Q: Is there homework?**
A: Light activities between sessions help reinforce learning. They're designed to be manageable.

**Q: What if I'm struggling?**
A: Please tell your trainer! We're here to support you and can adjust the pace or provide additional help.
