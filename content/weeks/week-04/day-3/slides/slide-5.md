# Week 4 Day 3 Slide 5: Practice Exercise - Decision Matrix
**Time: 25 minutes**

## Exercise
Apply a decision matrix to a real decision

## Steps
1. Define your options (3-4 choices)
2. List important criteria
3. Weight each criterion (importance)
4. Score each option against criteria
5. Calculate totals and compare

## Individual Task
Use this framework for a decision you're currently facing
