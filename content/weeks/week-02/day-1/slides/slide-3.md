# Week 2 Day 1 Slide 3: Break
**Time: 10 minutes**

## Break Instructions
Encourage reflection on skill development while resting.

### Physical Movement
- Stand and stretch
- Move away from learning materials
- Gentle breathing exercises

### Mental Processing
- Allow concepts to settle
- Notice which skills resonate most
- Brief reflection on current capabilities

### Return Time
Clearly state return time for group discussion.

## Trainer Modeling
Demonstrate professional break-taking as discussed in Week 1.

## Key Message
Skills develop better with regular processing breaks.
