# Week 2 Day 2 Slide 2: Communication Skills Framework
**Time: 40 minutes**

## Core Content

### Professional Communication Components
- **Clarity**: Clear message structure and language
- **Confidence**: Appropriate assertiveness and presence
- **Connection**: Building rapport and understanding
- **Consideration**: Awareness of audience and context

### Communication Channels
- **Written communication**: Emails, reports, documentation
- **Verbal communication**: Meetings, presentations, conversations
- **Non-verbal communication**: Body language, tone, presence
- **Digital communication**: Video calls, chat, collaborative platforms

### Practical Techniques
- Structure for clear messaging
- Active listening skills
- Professional tone development
- Feedback giving and receiving

## Key Message
Effective communication combines clear content with appropriate delivery.

## Teaching Approach
- Demonstrate techniques through examples
- Practice in low-stakes scenarios
- Build on individual communication strengths
- Address common communication challenges

## Learning Outcomes
- Understand professional communication framework
- Practice specific communication techniques
- Develop personal communication style
- Build confidence in workplace communication
